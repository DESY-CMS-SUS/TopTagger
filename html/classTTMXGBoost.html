Detailed Description

This module implements an interface to the OpenCV randomforest package for filtering top candidates. This module can either pass entries directly into the final top list, or filter entries out of the final top list if they do not pass the selection criteria.

Parameters
discCut(float) Minimum threshold for the TMVA discriminator for the candidate to pass the selection
modelFile(string) Path to the model file
NConstituents(int) Category of top to apply selection too (1 - monojet, 2 - dijet, 3 - trijet)
NCores(int) Number of cpu to allow XGBoost to use (default 1)
csvThreshold(float) Threshold on b-tag discriminator to be considered a b-jet.
bEtaCut(float) Requirment on |eta| for a constituent to be considered a b-jet
maxNbInTop(int) The maximum number of constituent jets which can be b-tagged for the candidate to be a final *
mvaVar[](string - array) MVA variable input names
