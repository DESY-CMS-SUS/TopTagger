Detailed Description

This module implements an interface to Tensorflow through python for filtering top candidates. This module places top candidates which pass the requirements directly into the final top list.

Parameters
discCut(float) Highest minimum discriminator threshold allowed (If discOffest is set > 1 and discSlope is positive then this serves as a basic discriminator threshold)
discOffset(float) Discriminator cut for zero pt top candidates
discSlope(float) Pt dependent slopt for discriminator cut
modelFile(string) Path to the model file
NConstituents(int) Type of constituent to apply selection to (1 - monojet, 2 - dijet, 3 - trijet)
inputOp(string) Input operation for the tensorflow graph
outputOp(string) Output operation of the tensorflow graph
csvThreshold(float) Threshold on b-tag discriminator to be considered a b-jet.
bEtaCut(float) Requirment on |eta| for a constituent to be considered a b-jet
maxNbInTop(int) The maximum number of constituent jets which can be b-tagged for the candidate to be a final top (set < 0 to disable)
mvaVar[](string - array) MVA variable input names
